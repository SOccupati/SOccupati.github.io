React Native Projects

Menu App

The Menu App is a React application that displays a restaurant menu with various categories and items. It utilizes React components and props to render the menu structure dynamically. One challenge I faced was implementing the category filtering functionality, which I solved by creating a reusable filter component and leveraging JavaScript's built-in array methods.

Todo App

This project is a Todo application built with React. It allows users to add, delete, and mark tasks as completed. I utilized React hooks and components to create a modular and reusable codebase. One challenge I faced was implementing the delete functionality, which I solved by passing a callback function from the parent component to the child component.

Exercise App

The Exercise App is a React application that helps users track their workout routines and exercises. It incorporates features such as exercise logging, progress tracking, and user authentication. One challenge I encountered was integrating a backend API for data storage and retrieval, which I overcame by using React's built-in fetch API and managing the application's state effectively.

Quiz App

This project is a Quiz application developed with React. It presents users with a series of questions and allows them to submit answers and view their scores. I focused on creating an engaging user interface and implementing dynamic question rendering and scoring logic. One challenge I faced was managing the application's state across multiple components, which I solved by using React's context API.

React Native Projects

React Native Weather App

This is a mobile application built with React Native that fetches and displays weather information based on the user's location. I utilized the Geolocation and OpenWeatherMap APIs to retrieve location data and weather details. The main challenge was integrating the APIs and handling asynchronous data fetching, which I overcame by using React hooks like useState and useEffect.
