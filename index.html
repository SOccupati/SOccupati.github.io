The Exercise App is a React application that helps users track their workout routines and exercises. It incorporates features such as exercise logging, progress tracking, and user authentication. One challenge I encountered was integrating a backend API for data storage and retrieval, which I overcame by using React's built-in fetch API and managing the application's state effectively.

This project is a Todo application built with React. It allows users to add, delete, and mark tasks as completed. I utilized React hooks and components to create a modular and reusable codebase. One challenge I faced was implementing the delete functionality, which I solved by passing a callback function from the parent component to the child component.

This project is a Flashcard application developed with React. It presents users with a series of flashcards and allows them to submit answers and view their scores for studying. I focused on creating an engaging user interface and implementing dynamic question rendering and scoring logic. One challenge I faced was managing the application's state across multiple components, which I solved by using React's context API.
